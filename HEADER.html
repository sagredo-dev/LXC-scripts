My LXC scripts for unprivileged containers

Handling unprivileged containers is not possible with default LXC programs, because they must be called by the user who owns them. This is my collection of wrapper scripts that I use both for privileged and unprivileged containers. In the latter case the owner will be dinamically retrieved.

More info at https://notes.sagredo.eu/lxc-scripts-280/wrapper-scripts-for-lxc-unprivileged-containers-258.html.
